# 프로젝트 개요

## 1.1 프로젝트 주제

본 프로젝트는 아두이노와 라즈베리파이를 기반으로 한 AIoT 기술을 활용하여, 아파트 주거 환경에서 발생하는 다양한 상태 정보를 수집·분석하고,웹 기반 인터페이스를 통해 스마트홈 설비를 통합적으로 관리·제어할 수 있는 시스템을 개발하는 것을 주제로 한다.

## 1.2 기획 배경 및 목적

현대 도시 주거의 대부분은 아파트 형태로 구성되어 있으며, 가구 유형으로는 1인 가구, 맞벌이 가구 가족단위 등 다양화 되고 있다. 이로 인해 집 내부 상태 원격으로 제어, 반복적인 생활 관리의 자동화, 단순한 제어를 넘어선 AI 관리 시스템 같은 요구사항 증가되고 있다.

본 프로젝트는 AI, 아두이노, 라즈베리파이를 결합하여 아파트 주거하는 환경을 제어하는 지능형 스마트홈 관리 시스템을 구현하는 것을 목표로 한다.
